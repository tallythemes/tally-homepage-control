Customize or Edit Home page contact for Tally Themes 

== Description ==

This plugin will enable home page content Control option. It will allow you to customize/edit predefined home page content for Tally Themes

== Installation ==

1. Upload the plugin to your WordPress site in the `/wp-content/plugins` folder or install via the WordPress admin.
2. Activate it from the Wordpress plugin admin screen.
3. After activation the plugin Go Tally Builder navigation to edit home page content.


== Frequently Asked Questions ==

= What does Tally Theme Setup do? =
It will allow you to customize/edit predefined home page content for Tally Themes

= Will it work on any theme? =
No, It will only work on supported theme of TallyThemes.com

= Why I need it? =
You need to edit home page content of our theme.
